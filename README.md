# EduHub 项目

## 项目简介
EduHub 是一个基于任务跟踪的智慧教育管理中心，旨在提供完整的教育管理功
能。

## 技术栈
- 后端框架: NestJS
- 数据库: PostgreSQL
- ORM: TypeORM
- 认证: JWT
- 测试框架: Jest

## 项目状态
项目已终止开发。
